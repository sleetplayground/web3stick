### DEV
I use a mixture of vite and parcel for this project.
```sh
bun install
bun run start # for parcel
bun run dev # for vite
bun run build # with vite
bun run clean

# web4 deploy
NEAR_SIGNER_KEY=ed25519:your_key_here
bun run web4_testnet
bun run web4_mainnet
```
